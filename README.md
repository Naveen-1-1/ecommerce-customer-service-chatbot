# E-commerce Customer Service Chatbot

An NLP-based chatbot designed to handle customer service queries for e-commerce platforms.

## Project Structure

| Notebook | Description |
|----------|-------------|
| `NLP_Project_Dataset_Analysis.ipynb` | Dataset exploration and preprocessing |
| `NLP_Project_Model.ipynb` | Core model training and development |
| `NLP_Project_End_to_End&Metrics.ipynb` | End-to-end pipeline and evaluation metrics |
| `NLP_Project_Additional_Analysis.ipynb` | Supplementary analysis |
| `NLP_Project_Large_Model_Testing.ipynb` | Large model experiments |
| `NLP_Project_Sparse_Query_Testing.ipynb` | Sparse query handling tests |

The `live-query-testing-tool/` directory contains the interactive chatbot demo.

## Requirements

- Python 3.x
- Jupyter Notebook
- GPU recommended (A100 used for training)
